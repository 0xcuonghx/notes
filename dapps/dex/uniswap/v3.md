# Concentrated liquidity

- The price of x in terms of y

```
p = y / x
```

- tick

```
p = 1.0001 ^ tick
```

- tick spacing: number of ticks to skip when the price moves

## sqrtPriceX96

```
sqrtPriceX96 = sqrt(P) * Q96
Q96 = 2^96

=> P = (sqrtPriceX96 / Q96)
```

## Tick

```
P = (sqrtPriceX96 / Q96) ^ 2 = 1.0001 ^ tick
```
