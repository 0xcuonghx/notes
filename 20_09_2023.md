# How Ethereum calculator smart contract address
## `CREATE` opcode
```
new_address = hash(sender, nonce)
```
## `CREATE2` opcode
```
new_address = hash(0xFF, sender, salt, bytecode)
```
## `CREATE3` opcode
```
new_address = hash(0xFF, sender, salt, KECCAK256_PROXY_CHILD_BYTECODE)
```
